### 安裝環境

poetry shell 虛擬環境
poetry install 下載 python 相應套件

### 執行環

make server 開啟伺服器

### 技術使用

前端：TailwindCSS, Alpine.js, HTMX

後端：Python, Flask

資料庫：SQLite

版本控制：Git

部署：AWS EC2, Nginx
